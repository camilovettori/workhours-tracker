# ⏱️ Work Hours Tracker – Progressive Web App (PWA)

A real-world work hours tracking application built to solve a practical problem in a live work environment.

## 🔗 Live Application
👉 https://workhours-tracker.onrender.com/

---

## 🚀 Key Features

- ⏱️ Clock in / Clock out / Break tracking  
- 📅 Work schedule and roster management  
- 💰 Real-time earnings calculation (hourly rate)  
- 🏦 Bank holiday handling  
- 📱 Progressive Web App (installable on mobile devices)  
- 💾 Local data persistence  
- 📊 Simple and intuitive user interface  

---

## 🛠 Tech Stack

### Backend
- Python  
- FastAPI  

### Frontend
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

### Database
- SQLite  

### PWA Technologies
- Service Worker  
- Web App Manifest  

### Deployment
- Render (Backend hosting)

---

## ⚙️ Architecture Overview

- REST API built with FastAPI  
- Frontend communicates using JSON-based requests  
- Local storage used for client-side data persistence  
- Lightweight architecture focused on performance and usability  

---

## 📌 Project Purpose

This application was developed to solve a real need within a workplace, allowing colleagues to track working hours, breaks, and earnings more efficiently.

The system is actively used in a real work environment, demonstrating practical problem-solving and the ability to deliver functional software solutions.

---

## 💡 Key Learning Outcomes

- Building REST APIs using FastAPI  
- Designing simple and efficient backend systems  
- Implementing PWA features for mobile usability  
- Handling real user requirements and feedback
